UI库类

# UI库类

# Layui

layui (opens new window) 是一套开源的 Web UI 解决方案，其内部采用的是自身经典的模块化规范，并遵循原生 HTML/CSS/JS 的开发方式，极易上手，拿来即用。其外在极简，却又不失饱满的内在，体积轻盈，组件丰盈，从核心代码到使用方法的每一处细节都经过精心雕琢，非常适合网页界面的快速开发。

# iView

iView (opens new window)，一套基于 Vue.js 的高质量 UI 组件库

# Element

Element (opens new window)，一套为开发者、设计师和产品经理准备的基于 Vue 2.0 的桌面端组件库

# Ant Design

基于 Ant Design (opens new window) 设计体系的 React UI 组件库，用于研发企业级中后台产品。

# Vant

Vant (opens new window)，轻量、可靠的移动端 Vue 组件库

# ThorUI

ThorUI (opens new window)，轻量、简洁、全面的移动端组件库。

# Naive UI

Naive UI (opens new window)，一个 Vue 3 组件库，比较完整，主题可调，使用 TypeScript，不算太慢，有点意思。

# uView

uView (opens new window) - 全面兼容nvue的uni-app生态框架 - uni-app UI框架。
