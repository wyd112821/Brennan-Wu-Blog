# Nuxt.js 中文网

Nuxt.js (opens new window) 是一个基于 Vue.js 的轻量级应用框架，可用来创建服务端渲染 (SSR) 应用，也可充当静态站点引擎生成静态站点应用，具有优雅的代码结构分层和热加载等特性。

# ECMAScript 6

《ECMAScript 6 入门教程》 (opens new window) 是一本开源的 JavaScript 语言教程，全面介绍 ECMAScript 6 新引入的语法特性。

《TypeScript中文网》 (opens new window) Typescript是Javascript类型的超集，它可以编译成纯Javascript。Typescript可以在任何浏览器、任何计算机和任务操作系统上运行，并且是开源的。

# 菜鸟教程

# Sass 中文网

# Sassmagic

SassMagic (opens new window) 是一个 Sass mixin 和函数的集合，以简化和改进我们的通用样式代码模式的实现。

# webpack 中文文档

webpack (opens new window) 是一个模块打包器。它的主要目标是将 JavaScript 文件打包在一起，打包后的文件用于在浏览器中使用，但它也能够胜任转换（transform）、打包（bundle）或包裹（package）任何资源(resource or asset)。
